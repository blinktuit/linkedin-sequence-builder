# LinkedIn Sequence Builder

A campaign automation tool for building and managing LinkedIn outreach sequences.

## Features

- Visual sequence builder with drag-and-drop interface
- Multiple campaign types (cold outreach, follow-up, event attendees, etc.)
- AI-powered actions (Perplexity research, message generation, classification)
- Conditional workflows with A/B testing
- Template library for quick campaign setup
- Lead import from CSV files

## Tech Stack

- **Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Routing:** React Router

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd linkedin-sequence-builder

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:8080`.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
